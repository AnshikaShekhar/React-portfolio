
🚀 Introduction

This is a personal portfolio website built using React, designed to showcase my skills, projects, and experience. The website features a modern UI, interactive components, and a responsive layout to ensure compatibility across devices and browsers.

🛠️ Tech Stack

Frontend: React, HTML5, CSS3, JavaScript

Styling: Tailwind CSS / CSS Modules

Build Tool: Vite

Other Tools: Git, GitHub, React Router

🎯 Features

Dynamic Project Showcase: Displays projects with descriptions, tech stack, and GitHub links.

Skills Section: Highlights my technical expertise in frontend and backend technologies.

Contact Form: Allows visitors to send messages directly.

Responsive Design: Ensures a seamless experience across all screen sizes.
